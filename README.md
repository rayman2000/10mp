# 10 Minute Pokemon (10mp)

A collaborative Pokemon Fire Red experience where players take turns playing for exactly 10 minutes.

## Setup

1. `npm install`
2. Add Pokemon Fire Red ROM to `public/emulator/pokemon-firered.gba`
3. `npm start`
